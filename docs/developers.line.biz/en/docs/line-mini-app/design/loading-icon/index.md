# Loading icon

Loading icons are a recommended UI element in LINE MINI App.

Download the spinner (svg file) and use it as a loading icon.

- Light mode spinner ([Download](https://developers.line.biz/media/line-mini-app/LINE_spinner_light.svg))
- Dark mode spinner ([Download](https://developers.line.biz/media/line-mini-app/LINE_spinner_dark.svg))

The size of the spinner is 30x30 pixels.
Center align the spinner.
