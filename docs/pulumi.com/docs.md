# **Pulumi Docs**[](#pulumi-docs)

Pulumi is an [open source](https://github.com/pulumi/pulumi) platform for building, deploying, and managing cloud infrastructure using your favorite [programming languages](/docs/iac/languages-sdks/). Manage infrastructure, secrets, and configuration with a unified workflow across any cloud.

[Get Started](/docs/get-started/)
